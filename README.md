# 港股 IPO / 二级交易投资决策系统

本版本为“模型可信度与展示口径增强版”。系统覆盖：

- A1/上市申请阶段未上市项目
- 招股期参与决策
- 暗盘与首日交易
- 所有 2024 年后上市公司的二级市场交易状态机，不限 180 日
- 解禁与供给压力
- 评分标准、量化路径定义与回测有效性验证

## 本版重点优化

1. 决策总览按阶段显示不同评级：未上市看项目质量，已上市看二级交易评分。
2. 决策总览筛选器改为匹配当前页面实际评级字段。
3. A1 项目观察池新增评分置信度与日期校验。
4. 招股期参与决策拆分为“当前招股/待上市项目”和“历史招股样本复盘”。
5. 二级市场交易状态机新增行情新鲜度，显示正常、轻微滞后、明显滞后、缺失。
6. 路径定义和招股期评判标准全部量化展示。
7. 回测页面增加结论先行与表格解释，并将主要收益/胜率格式化为百分比。
8. TradingView 链接改为嵌入代码和公司名称，点击代码或名称即可打开图表。

## 本地运行

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
