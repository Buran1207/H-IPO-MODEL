# Step 7 评分逻辑说明

## 四阶段评分

1. **A1 预筛分**：用于递表、临时代码、通过聆讯、未招股公司。
2. **一级参与分**：用于打新、基石、锚定额度判断。
3. **二级交易分**：用于暗盘、首日、上市后0-180D状态机。
4. **综合分**：已上市股票提高二级权重，未上市股票提高A1和一级权重。

## 风险标签

- 孖展极度拥挤
- 公开认购过热
- 一手中签率低
- 基石阵容较多
- 关注基石锁定期
- 上市后回撤大
- 首日情绪后续不足
- 免费行情缺失 / 行情样本不足

## 解释边界

该模型只给研究和交易建议，不接实盘，不自动下单。高分不等于一定买入，低分不等于永远不看；最终应结合行业研究、估值、额度、流动性和组合仓位。
